1. 현재 수사 보드

초록: 새 단서 추가 주황: 내용 또는 상태 변경 파랑: 보드 종류 교체 빨강: 사라진 단서 위치

2. 다음 가설 보드

오른쪽 작은 웹페이지는 아직 반영되지 않은 가설 보드입니다. 비교 규칙에 맞지 않는 공백 텍스트와 주석은 자동으로 제외됩니다.

3. 수사 기록 수정하기

코드를 직접 고치지 않고, 사건명과 단서 카드만 바꿔도 오른쪽 가설 보드가 먼저 달라집니다.
