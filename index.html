1. 현재 수사 보드

초록: 새 단서 추가 주황: 내용 또는 상태 변경 파랑: 보드 종류 교체 빨강: 사라진 단서 위치

2. 다음 가설 보드

오른쪽 작은 웹페이지는 아직 반영되지 않은 가설 보드입니다. 비교 규칙에 맞지 않는 공백 텍스트와 주석은 자동으로 제외됩니다.

3. 수사 기록 수정하기

코드를 직접 고치지 않고, 사건명과 단서 카드만 바꿔도 오른쪽 가설 보드가 먼저 달라집니다.

사건명 48자, 사건 상태 24자, 수사 메모 120자, 용의자 이름 20자, 용의자 메모 60자, 단서 카드는 한 줄당 40자까지 입력할 수 있습니다.

수사 보드에 실제로 반영되는 HTML 조각입니다. 이 코드를 직접 고치면 오른쪽 가설 보드가 바로 다시 계산됩니다.

현재 수사 보드 HTML을 불러오는 중입니다.
